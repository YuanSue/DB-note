Data Stage

現今主流的ETL工具

IBM Infosphere Data stage

- 適合大規模資料整合
- 支援DB:僅支援 DB2、Oracle、MS SQL
- 強大的 parallel processing engine，效能佳

Trinity

- 由Teradata團隊分家出來開發的ETL工具
- 支援DB:支援 DB2、Oracle、MS SQL、Postgres、Teradata...etc
- 排程功能強，但ETL圖型化流程較弱

Informatica

- 資料源整合支援多
- 同時支援傳統 RDB 與現代雲端平台

IBM Infosphere Data stage

- 設計ETL
- 整合專案的工具: 能整合data ware house 、data mart、 system migration
- 能創建、運行集集監控專案
- Import, export, create, and manage metadata for use within jobs
note: Metadata=>描述"其他資料資訊"的資料(紀錄欄位的定義)
